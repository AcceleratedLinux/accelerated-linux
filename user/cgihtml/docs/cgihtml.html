cgihtml

version 1.65

a set of CGI and HTML routines written in C
by Eugene Eric Kim, <eekim@eekim.com>

Named Top 2 CGI Programming Tools by PC Computing (August 1996).

This documentation is also available at http://www.eekim.com/software/cgihtml/cgihtml.html

Where to Find cgihtml

You can download cgihtml (gzipped, UNIX commpressed, or PKZipped) from one of the following sites:

eekim.com FTP Site
ftp://ftp.eekim.com/users/eekim/cgihtml/
Harvard Computer Society FTP Site
ftp://hcs.harvard.edu/pub/web/tools/cgihtml/

Useful information about cgihtml is available at the cgihtml home page, located at http://www.eekim.com/software/cgihtml/.
