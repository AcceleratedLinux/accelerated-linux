   Данный  комплект   представляет  собой   локализованное  дополнение   к
   архиватору ARJ v @COUNTERPARTS  для DOS. Наличие  ARJ для  DOS необязательно, но
   рекомендуется, поскольку файлы документации не входят в поставку.

   ВВЕДЕНИЕ

      В этом файле  описаны  функциональные  отличия данной  реализации от
      исходной  версии  ARJ для DOS. Перед  его  прочтением  рекомендуется
      ознакомиться с общим файлом README.


   ИЗМЕНЕНИЯ В ЛОКАЛИЗОВАННОЙ ВЕРСИИ

      *  Поддерживаются (не искажаются при внесении изменений) расширенные
         атрибуты в архивах, созданных ARJ/2.

      *  Локализованная версия  ARJ для DOS собрана с  помощью компилятора
         Microsoft C/C++ v 8.0, в то  время, как  американская - с помощью
         Borland C++ v 3.1. Это  накладывает ограничения  на совместимость
         модулей шифрования, в  частности, не  рекомендуется  использовать
         вместе с  русифицированным ARJ  модуль от американской  версии, и
         наоборот.

      *  В  многозадачных  системах   используется более  интеллектуальный
         механизм  проверки заблокированных файлов, с запросом  действия у
         пользователя.
