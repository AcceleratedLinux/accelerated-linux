Список часто используемых команд и ключей.  По ARJ -? выдается полный список.

Запуск:    ARJ <команда> [-<ключ> [-<ключ>...]] <архив> [<файл(ы)>...]
Примеры :  ARJ a -e archive, ARJ e archive, ARJ l archive *.doc
<Команды>
 ac: добавить очередную страхов. копию   l: вывести список содержимого
  a: добавить файлы к архиву             m: переместить файлы в архив
  c: снабдить файлы комментариями        t: проверить целостность архива
  d: удалить файлы из архива             u: обновить файлы в архиве
  e: распаковать архив                   v: вывести детальный список файлов
  f: обновить существ. в архиве файлы    x: распаковать файлы с путями
<Ключи>
  c: не проверять дату/время             r: пройти по подкаталогам
  e: исключить пути из имен файлов       u: обновить файлы (по дате)
  f: обновиь существующие файлы          v: включить многотомный режим
  g: зашифровывать с паролем             w: назначить рабочий каталог
  i: отключить индикатор состояния       x: исключить выбранные файлы
  m: выбрать метод: 0, 1, 2, 3 или 4     y: "да" в ответ на все вопросы
  n: только новые (несуществ.) файлы    hk: добавлять страховочную запись
