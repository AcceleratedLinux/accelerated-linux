Все права защищены. REARJ - утилита для преобразования архивов к формату ARJ.
Запуск:  REARJ [параметры] <файлы, !списки или маски файлов>
  Параметры (могут задаваться до и после имен файлов)
  /+             игнорировать переменную окружения REARJ_SW
  /a[расширение] конвертировать вложенные архивы ("*" - все форматы)
  /b<команда>    выполнить команду перед распаковкой
  /c<команда>    выполнить команду перед пересчетом распакованных файлов
  /d             удалять исходные архивы
  /e             игнорировать ошибки
  /f             конвертировать архивы на гибких дисках
  /g             работа в холостом режиме
  /i[имя_файла]  проверить целостность программы
  /j             не сохранять архив, если преобразование убыточно
  /k             проставлять текущую дату на архив
  /l[имя_файла]  вести журнал транзакций (по умолчанию - REARJ.LOG)
  /m[дата]       выбрать архивы, созданные до ГГММДДЧЧММСС
  /n[дата]       выбрать архивы, созданные ГГММДДЧЧММСС или позднее
  /o             перезаписывать существующие архивы
  /p             игнорировать длинные имена в среде Windows 95
  /q             запрашивать подтверждение для каждого архива
  /r             осуществлять рекурсию по поддиректориям
  /s             не проверять количество и общий размер файлов
  /t<расширение> конвертировать в другие типы архивов
  /u[bak]        обновлять архивы с запасными копиями (по умолчанию - BAK)
  /v             выполнить определенную команду на распакованных файлах
  /w<путь>       указать рабочую директорию
  /x[!]<файл>    исключить файл, файлы по маске или файлы из списка
  /y[текст]      удалить полученный архив и записать текст в журнал
  /z             сымитировать выполнение
Пример:  REARJ *.* /r /d    Сконвертировать все архивы в ARJ, включая
                            архивы в поддиректориях, исходные архивы удалить
Переменная окружения REARJ: SET REARJ_SW=/l /we:\temp
