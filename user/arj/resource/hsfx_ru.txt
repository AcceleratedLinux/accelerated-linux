
Команды:
e: распаковать файлы             v: вывести детальный список файлов
l: вывести список файлов         x: распаковать файлы с путями (по умолчанию)
t: протестировать архив

Параметры:
a: показывать ANSI-комментарии   m: создавать директории
b: задать путь с клавиатуры      n: только новые файлы
c: не проверять время            o: перезаписывать существующие файлы
d: запустить программу ARJDISP   p: сверять имя пути
f: обновить существующие файлы   s: не проверять архив
g: расшировать с паролем         u: обновить файлы
i: не выводить индикатор         w: не запрашивать подтверждение на распаковку
j: обрабатывать длинные имена    y: "да" в ответ на все вопросы
k: выдавать детальные сообщения  !: выполнить внешнюю команду
