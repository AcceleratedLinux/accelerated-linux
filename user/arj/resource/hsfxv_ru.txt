Команды:
e: распаковать файлы             v: вывести детальный список файлов
l: вывести список файлов         x: распаковать файлы с путями (по умолчанию)
t: протестировать архив

Параметры:
a: не проверять архив            s: включение разделенного доступа к файлам
b: задать путь с клавиатуры         (предназначение ключа изменилось)
c: не проверять время            u: обновить файлы
d: проверять место на диске      w: записать в файл с заданным именем
f: обновить существующие файлы   y: "да" в ответ на все вопросы
g: расшифровать с паролем        z: "да" в ответ на вопросы:
h: вывести эту справку              a: дополнить?,      c: создать директорию?
i: не выводить индикатор            k: мало места,      m: распаковать?
j: обрабатывать длинные имена       n: переименовать?,  o: перезаписать?
k: игнорировать настройки           r: (очистка ввода), v: следующий том?
m: перезаписывать RO-файлы       #: начинать распаковку с N-го диска
n: только новые файлы            $: распаковывать метку тома
o: указать модуль шифрования     @: выдать параметры запуска
p: сверять имя пути              !: выполнить внешнюю команду
r: игнорировать повреждения
