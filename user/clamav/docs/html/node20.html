clamav-milter

ClamAV 0.95 includes a new, redesigned clamav-milter. The most notable difference is that the internal mode has been dropped and now a working clamd companion is required. The second important difference is that now the milter has got its own configuration and log files. To compile ClamAV with the clamav-milter just run ./configure --enable-milter and make as usual. Please consult your MTA's manual on how to connect it with the milter.
