Archives and compressed files

The following archive and compression formats are supported by internal handlers:
  • Zip (+ SFX)
  • RAR (+ SFX)
  • Tar
  • Gzip
  • Bzip2
  • MS OLE2
  • MS Cabinet Files (+ SFX)
  • MS CHM (Compiled HTML)
  • MS SZDD compression format
  • BinHex
  • SIS (SymbianOS packages)
  • AutoIt
  • NSIS
